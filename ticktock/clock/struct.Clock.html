Waits for the next clock tick.

Will wait until the next tick and return the current tick count. If the clock has become nonsensical (for example due to do a change of the system clock into the past/future, will return None.

fn iter(&mut self) -> ClockIter

Creates a clock iterator.

The iterator will iterate forever, calling wait_until_tick on each iteration. If an error occurs, it will reset the clock using restart, then resume operatior.

Returns the current tick number on each iteration.
